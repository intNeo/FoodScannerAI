"# FoodScannerAI"

Распознавание блюд по фотографии с помощью глубокого обучения.



Приложение определяет категорию блюда по загруженному изображению и показывает 3 наиболее вероятных класса с уверенностью модели.

Реализовано сравнение 5 архитектур нейронных сетей на датасете Food-101. Лучшая модель — \*\*EfficientNet-B0\*\* (точность на валидации 69.19%).
